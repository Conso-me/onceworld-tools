# OnceWorld ステータス計算式・ダメージ計算式

Wiki確認済みの計算式をまとめたリファレンス。

---

## ステータスレベルスケーリング

```
scaledStat = Math.floor(baseStat * ((level - 1) * 0.1 + 1))
```

## HP計算

```
HP = scaledVit * 18 + 100
```

## ダメージ計算

### 物理 / 魔弾 (ペット・敵の魔法攻撃)

```
baseDamage = max((ATK or INT) * 1.75 - effectiveDef, 0) * 4 * 属性倍率
物理の場合: effectiveDef = DEF + floor(M-DEF / 10)
魔弾の場合: effectiveDef = M-DEF + floor(DEF / 10)
```

### 主人公魔法 (魔攻)

```
baseDamage = max((INT + 解析書INT) * 1.25 * 魔法倍率 - effectiveDef, 0) * 4 * 属性倍率
effectiveDef = M-DEF + floor(DEF / 10)
解析書INT = 解析書数 * (1 + 解析解析書数 * 0.1)
```

### 乱数・クリティカル

```
最終ダメージ = baseDamage * 乱数(0.9〜1.1) * クリティカル(2.5) * 多段回数
最小ダメージ: baseDamage <= 0 の場合、1〜9 のランダムダメージ
```

### 多段攻撃 (SPD依存)

主人公魔法(魔攻)は常に1回。それ以外:

| SPD範囲 | 多段回数 |
|---------|---------|
| 0 - 2,999 | 1 |
| 3,000 - 9,999 | 2 |
| 10,000 - 29,999 | 3 |
| 30,000 - 99,999 | 4 |
| 100,000+ | 5 |

### LUCK (命中・回避)

```
命中必要LUCK = ceil(敵LUCK * 命中倍率)
回避必要LUCK = max(ceil(敵LUCK * 回避倍率), 100)
```

### MOV

MOVはレベルスケーリングしない（baseMov固定）。

## 経験値スケーリング

```
scaledExp = max(floor(level^1.1 * 0.2), 1) * baseExp
```

## 被ダメ無効ライン

ダメージ計算結果が0以下 → 被ダメージは1〜9のランダム

```
物理無効化: DEF + floor(M-DEF / 10) >= 敵ATK * 1.75
魔法無効化: M-DEF + floor(DEF / 10) >= 敵INT * 1.75
```
